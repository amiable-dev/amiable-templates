## Ways to Contribute

1. **Add a new template** - Share your Railway deployment configuration
2. **Improve documentation** - Fix typos, add examples, clarify instructions
3. **Report issues** - Found a bug or broken link?

## Getting Started

```bash
# Clone the repository
git clone https://github.com/amiable-dev/amiable-templates.git
cd amiable-templates

# Install dependencies
pip install -r requirements.txt

# Run locally
mkdocs serve
```

Visit http://localhost:8000 to preview the site.
