# Announcements

This directory contains draft social media announcements for project milestones.

## Structure

Each announcement has platform-specific versions:

- `*-twitter.md` - Twitter/X (280 char limit)
- `*-linkedin.md` - LinkedIn (professional tone)
- `*-devto.md` - Dev.to (developer community)
- `*-hackernews.md` - Hacker News (Show HN format)

## Process

1. Draft announcements after completing a milestone
2. Review with LLM Council (High tier)
3. Incorporate feedback
4. Post to platforms

## Guidelines

- Keep messages short and informative
- Target engineering community
- Include relevant links
- Use platform-appropriate tone
